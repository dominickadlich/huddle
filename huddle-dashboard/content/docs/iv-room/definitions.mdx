## Pinnacle Pump
: This is a nine-port compounder used in pumping single or a mixture of fluid bags to obtain desired volume and concentration. 

## Repeater Pump
: This is a single port compounder used to facilitate automated fluid transfer of single fluid bag to obtain desired volume.   

## Exactamix 2400 Pro
: This is a TPN compounder with twenty-four ports designed to accurately pump ingredients that make a TPN mixture. The minimum volume it can pump is 0.2mL. 

## MedKeeper (PharmacyKeeper)
: Cloud-based software that provides guided compounding workflow. Used for documentation and verification of compounding as well documentation of activities
